# FinGuard Smart Aquaculture Monitoring System

## Overview

FinGuard is an Arduino-based aquaculture monitoring system developed during ASEM Embedded Systems Training.

## Features

- Sensor Monitoring
- Bluetooth Low Energy (BLE)
- Real-Time Data Collection

## Technologies

- Arduino
- BLE
- Embedded C
